# music

### 这是一个用vue仿写的网易云音乐项目，主要实现了播放功能、搜索功能以及个性推荐页面、歌单页面、歌单详情页、主播电台页面、排行榜页面、歌手页面、歌手详情页、最新音乐页面。

这个项目主要是用来练手同时巩固知识点，还存在许多的瑕疵，比如代码冗余。所以还有很大的改进空间，以后慢慢改进。
